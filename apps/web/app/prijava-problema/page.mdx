## Kako prijaviti problem?

| Korak | Opis |
|-------|------|
| **1.** | Otvorite [Geoforum sustav](http://localismarket.gdi.net/Forum/opcina-veliki-bukovec/Issues#/allItems) |
| **2.** | Označite lokaciju problema na karti |
| **3.** | Odaberite vrstu problema |
| **4.** | Opišite problem i priložite fotografiju (opcionalno) |
| **5.** | Pošaljite prijavu |
